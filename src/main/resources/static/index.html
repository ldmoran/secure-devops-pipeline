DevSecOps Pipeline

Este proyecto universitario implementa un flujo de trabajo automatizado y seguro que integra inteligencia artificial para salvaguardar la integridad del código fuente.

  • GitHub Actions: Orquestación e integración continua automatizada ante cualquier evento en el repositorio.
  • Análisis de Seguridad con IA: Evaluación estática y dinámica mediante un modelo de Machine Learning Random Forest entrenado por los estudiantes para clasificar código o payloads sospechosos.
  • Bloqueo Automático: Restricción inmediata y rechazo de Pull Requests que presenten vulnerabilidades detectadas por el modelo.
  • Notificaciones por Telegram: Alertas instantáneas dirigidas a un bot de Telegram informando el estado de los análisis del pipeline.
  • Despliegue Continuo (CD): Publicación automatizada del servicio backend e infraestructura directamente en la plataforma Render tras superar satisfactoriamente los filtros de seguridad.
